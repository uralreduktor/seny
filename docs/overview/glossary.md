# Глоссарий терминов и аббревиатур

## Общие термины

### Тендер
Процедура закупки товаров, работ или услуг, проводимая заказчиком в соответствии с законодательством о контрактной системе.

### ТКП (Технико-коммерческое предложение)
Документ, содержащий технические характеристики предлагаемого оборудования и коммерческие условия поставки.

### ЭТП (Электронная торговая площадка)
Интернет-платформа для проведения электронных торгов и закупок.

### ЕИС (Единая информационная система)
Единая информационная система в сфере закупок (zakupki.gov.ru) — государственная платформа для размещения информации о закупках.

### Сбербанк-АСТ
Электронная торговая площадка Сбербанк-АСТ для проведения коммерческих и государственных закупок.

### Росэлторг
Федеральная электронная торговая площадка для государственных и корпоративных закупок.

## Технические термины

### RAG (Retrieval-Augmented Generation)
Подход к генерации текста с использованием языковых моделей, дополненный поиском по базе знаний для повышения точности ответов.

### Embeddings (Эмбеддинги)
Векторные представления текста или данных в многомерном пространстве, используемые для семантического поиска.

### OCR (Optical Character Recognition)
Оптическое распознавание символов — технология преобразования изображений текста в машиночитаемый формат.

### pgvector
Расширение PostgreSQL для хранения и поиска векторных данных (embeddings).

### Vector Search (Векторный поиск)
Поиск по семантическому сходству с использованием векторных представлений данных.

## Бизнес-термины

### Позиция
Отдельный товар, работа или услуга, указанная в тендерной документации.

### Расчёт стоимости
Документированный процесс определения стоимости позиции с учётом материалов, трудозатрат, накладных расходов и маржи.

### Шаблон расчёта
Сохранённая методика расчёта стоимости для типовых позиций или технологий.

### Технологический маршрут
Последовательность операций производства изделия с указанием оборудования, времени и трудозатрат.

### КД (Конструкторская документация)
Комплект документов, содержащий чертежи, спецификации и другую техническую информацию об изделии.

### Редуктор
Механическое устройство для передачи и преобразования крутящего момента, изменения угловых скоростей и вращательных движений.

### Мотор-редуктор
Агрегат, состоящий из электродвигателя и редуктора, объединённых в единый корпус.

### Шестерня
Зубчатое колесо, передающее вращательное движение в зубчатой передаче.

## Роли пользователей

### Админ
Пользователь с полным доступом к системе, включая управление пользователями, настройки системы и конфигурацию.

### Руководитель
Пользователь с правами просмотра аналитики, назначения ответственных и контроля выполнения задач.

### Инженер
Основной пользователь системы, выполняющий расчёты, создающий ТКП и работающий с базой знаний.

### Чтение-only
Пользователь с правами только на просмотр данных без возможности редактирования.

## Форматы файлов

### DWG
Формат файлов чертежей AutoCAD.

### STEP
Стандарт обмена данными о продукте (ISO 10303) для обмена 3D-моделями.

### PDF
Portable Document Format — формат электронных документов.

## Интеграции

### КриптоПро
Российская система криптографической защиты информации, используемая для электронной подписи.

### Рутокен
Аппаратные токены для электронной подписи и аутентификации.

## Метрики и показатели

### Uptime
Время доступности системы, выраженное в процентах от общего времени работы.

### Точность OCR
Процент правильно распознанных символов или слов из изображения.

### Точность RAG
Процент релевантных ответов, полученных при использовании RAG-поиска.

### Время отклика
Время между отправкой запроса и получением ответа от системы.
